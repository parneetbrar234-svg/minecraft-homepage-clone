# Minecraft-Homepage-Clone

This project is a recreated homepage design of the official Minecraft website built using HTML and CSS. The aim of this project was to practice frontend web development by recreating a real website interface and improving styling and layout skills.

## About the Project
The website is inspired by the original Minecraft homepage and includes multiple sections similar to the real website design. The project focuses on layout creation, styling, positioning, and recreating a modern gaming website interface using only HTML and CSS.

## Features
- Navigation bar with menu items
- Minecraft themed homepage design
- Game cards with pricing and buttons
- Launcher features section
- Styled buttons and hover effects
- Multiple content sections
- Dark themed gaming UI

## Technologies Used
- HTML
- CSS

## What I Learned
Through this project, I practiced:
- Building webpage layouts
- Using Flexbox for alignment
- Styling sections and cards
- Working with images and backgrounds
- Creating responsive-looking UI components
- Improving frontend design skills

## Purpose
The main purpose of this project was to strengthen my frontend development basics by recreating a real-world website design and understanding how modern landing pages are structured.
